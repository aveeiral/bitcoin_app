This application is a bitcoin dashboard in which net_profit and loss in rereshed on selected bitcoins (XRP,BITCOIN, EOS and Ethereum).
The API is generated from CoinMarketCap API that shows the rate of several crptocurrencies

libraries used in this application :
1. json
2. requests
3. numpy
4. tkinter

Just run the "main.exe" file.
The red colour shows that currently hyaving loss with that coin.
The green colour shows that currently having profit with that coin.

It is also connected to a sql database 

Website used : https://coinmarketcap.com/
Website_API : https://pro-api.coinmarketcap.com/v1/cryptocurrency/listings/latest?start=1&limit=300&convert=INR&CMC_PRO_API_KEY=3e0a0b55-350c-4e1a-8828-83e8f7efdb67

P.S. The data refreshes every 20 seconds so you have to bear a little with the app.
